- Login: Single column, centered
- Forms: Slightly wider cards
- Font sizes: Medium (text-4xl)

- Padding: py-8 (32px)

### Mobile (<640px)
- All: Single column, full-width
- Logo: Smaller, centered at top
- Font sizes: Smaller (text-3xl)
- Cards: Full-width with px-4 padding
- Buttons: Full-width
