## 🎬 Animation Details

### Hero Section Animations
```
┌─────────────────────────────────────┐
│  ⭕ Orb 1 (Top-Left)                │
│     • Moves in X, Y, Scale          │
│     • 20s infinite loop             │
│     • Easing: easeInOut             │
│                                     │
│           ⭕ Orb 2 (Bottom-Right)   │
│              • Different pattern    │
│              • 25s infinite loop    │
│              • Blue color           │
└─────────────────────────────────────┘

Content Fade-In Timeline:
├─ 0.0s: Badge appears
├─ 0.1s: Heading appears
├─ 0.2s: Description appears
├─ 0.3s: Event details appear
├─ 0.4s: Buttons appear
└─ 0.6s: Scroll indicator appears

Scroll Indicator:
  ↓ Bounces up and down continuously
  • 2s loop
  • Y: 0 → 10px → 0
```

### Event Info Animations
```
Cards Stagger Animation:
┌──────┐     ┌──────┐     ┌──────┐     ┌──────┐
│Card 1│ →   │Card 2│ →   │Card 3│ →   │Card 4│
└──────┘     └──────┘     └──────┘     └──────┘
   0ms        100ms        200ms        300ms
   ↑            ↑            ↑            ↑
Fade-in + Slide-up from Y: 20 to Y: 0

Hover Effects:
• Icon: Scale 1.0 → 1.1
• Card: Shadow-md → Shadow-xl
• Duration: 300ms ease-smooth
```

### Statistics Animations
```
Counter Animation:
  0 ──────────────────> 2000+
  │                       │
  │  Smooth easing       │
  │  2 seconds           │
  └──────────────────────┘

Trigger: IntersectionObserver
• Animates when 50% visible
• Only animates once per page load
• Uses Framer Motion useMotionValue

Background Orbs:
⭕ Orb 1: 8s pulse (scale + opacity)
⭕ Orb 2: 10s pulse (scale + opacity)
```
